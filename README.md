# 🚗 Java Swing Car Game

A simple **2D Car Racing Game** developed using **Java Swing**. Control your car, avoid incoming obstacle cars, and score points while the game gets progressively faster.

## ✨ Features

* 🚗 Smooth car movement
* 🛣️ Scrolling road animation
* 🚙 Random obstacle cars
* 💥 Collision detection
* 📈 Increasing difficulty
* 🏆 Score tracking
* 🔄 Restart with **R** key

## 🛠️ Technologies

* Java
* Java Swing
* AWT Graphics
* OOP

## 🎮 Controls

* ⬅️ Left Arrow – Move Left
* ➡️ Right Arrow – Move Right
* ⬆️ Up Arrow – Move Up
* ⬇️ Down Arrow – Move Down
* **R** – Restart Game

## ▶️ Run the Project

```bash
javac *.java
java CarGameLauncher
```
